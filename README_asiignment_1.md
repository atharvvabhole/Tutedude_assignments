# Python Assignment 1

This repository contains two introductory Python scripts created for Assignment 1. The programs demonstrate basic input/output handling, arithmetic operations, and string formatting in Python.

## Files Included

* `Assignment_1_task_1.py`
* `Assignment_1_task_2.py`

## Task Descriptions

### Task 1: Basic Arithmetic Calculator (`Assignment_1_task_1.py`)
This program acts as a simple calculator. 
* It prompts the user to input two integer numbers.
* It calculates the addition, subtraction, multiplication, and division of the two provided numbers.
* Finally, it prints each result on a new line.

### Task 2: User Greeting (`Assignment_1_task_2.py`)
This program demonstrates string manipulation and concatenation.
* It prompts the user to enter their first name.
* It prompts the user to enter their last name.
* It uses Python f-strings to format the inputs into a personalized greeting: "Hello, [First Name] [Last Name]! Welcome to the Python Program".

## How to Run

To run these scripts, you will need Python installed on your machine. Open your terminal or command prompt, navigate to the folder containing these files, and run the following commands:

**To run Task 1:**
python Assignment_1_task_1.py

**To run Task 2:**
python Assignment_1_task_2.py
